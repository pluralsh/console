# Exa tool setup

Use this guide to setup your Exa integration

## Plural Cloud caveat

If you're using **Plural Cloud**, we provide a built-in Exa API key for Workbench use. In that case, you can leave `API key (optional)` blank unless you want to override with your own key.

## 1. Create an Exa API key

1. Go to the Exa dashboard API keys page: https://dashboard.exa.ai/api-keys
2. Create a new key and copy it immediately.
3. (Optional) Add a key name and rate limit for safer usage.

## 2. Fill the Workbench tool form

- `API key (optional)`: your Exa API key.
- If left empty, Workbenches can use a server-side Exa key when available.
