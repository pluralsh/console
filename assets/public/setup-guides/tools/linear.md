# Linear tool setup

Use this guide to fill `Access token`.

## 1) Create a Linear API key

In Linear:
1. Open **Settings -> Account -> Security & Access**.
2. Create a personal API key.
3. Copy and store the key immediately.

## 2) Restrict scope for least privilege

- Prefer a token with read scope only when available for your workflow.
- Restrict team/resource access during key creation if your workspace supports it.
- Use a dedicated integration account instead of a personal admin account.

## 3) Fill the Workbench tool form

- `Access token`: the Linear API key value
