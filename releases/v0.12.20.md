# v0.12.20

## New Features 🎉

* **Console release notes script** ([#3465](https://github.com/pluralsh/console/pull/3465)) by @JohnBlackwell

  > A new script has been introduced to automate the generation of release notes for the Plural Console, leveraging GitHub's API and an LLM to create user-friendly summaries of merged pull requests. This enhancement streamlines the release process by providing detailed patch notes that categorize changes into breaking changes, new features, bug fixes, and dependency updates, ensuring users are well-informed about updates and improvements. Additionally, the script includes a fallback mechanism to use PR descriptions when the LLM is not available, maintaining consistency in communication.

* **Compatibility Matrix Update 2026-05-09** ([#3535](https://github.com/pluralsh/console/pull/3535)) by @app/github-actions

  > The compatibility matrix has been updated to reflect the new chart version 9.5.13 for Argo CD, ensuring users have access to the latest features and fixes. This update is crucial for maintaining compatibility with Kubernetes versioning and enhances the overall stability of deployments using Argo CD.

* **Fix ai proxy chart** ([#3536](https://github.com/pluralsh/console/pull/3536)) by @michaeljguarino

  > The ai-proxy chart has been updated to version 0.2.6, introducing a new `PROVIDER_TOKENS` secret for enhanced configuration. Additionally, the AI provider has been switched from Bedrock to OpenAI, ensuring that the integration points to the correct base URL for OpenAI services. This change improves the functionality and reliability of AI-related features within the Plural Console.


## Bug Fixes 🐛

* **Minor tweaks** ([#3537](https://github.com/pluralsh/console/pull/3537)) by @michaeljguarino

  > The ai-proxy Helm chart has been updated to version 0.2.7, introducing a checksum annotation for configuration secrets in the deployment template. Additionally, the naming of a secret key has been standardized from `PROVIDER_TOKENS` to `PLRL_PROVIDER_TOKENS`, enhancing clarity and consistency in secret management. These changes improve the reliability of deployments and facilitate better tracking of configuration changes.

* **Prep hotfix release** ([#3538](https://github.com/pluralsh/console/pull/3538)) by @michaeljguarino

  > The hotfix release updates the Plural Console to version 0.6.38, introducing new AI model defaults for various providers, including OpenAI, Azure, and Anthropic. This enhancement allows users to leverage improved AI capabilities by automatically configuring optimal model settings, streamlining the integration of AI functionalities within their Kubernetes workflows. Additionally, the release includes updates to the secrets configuration, ensuring better management of sensitive information.

* **Fix workbench job sidebar logic** ([#3539](https://github.com/pluralsh/console/pull/3539)) by @michaeljguarino

  > The sidebar logic in the workbench has been refined to ensure that the side panel only appears when users are not viewing job details. This enhancement streamlines the user interface, allowing for a more focused experience when managing workbench jobs, thereby reducing clutter and improving usability.
